pallet.crate.app-deploy.lein documentation

Resolve artifacts for deploying via lein command line.

Each artifact map should provide at least a :project-path entry. The :project-path entry may contain a "%s", which will be replaced with the project's version string.

{:project-path "target/app-%s-standalone.jar" :path "/opt/myapp/simple-app.jar" ; Path to install to ; (relative or absolute) :unpack false} ; flag to unpack the artifact
